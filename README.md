# Python-Crash-Course

## Python从入门到实践

---
- 这个项目是《Python从入门到实践》这本书的第18～19章内容的代码实现
- 因为作者书写此书的时间原因，使用的是Django1.x，本项目采用的是Django2.x
- 针对版本不同使用的不同点，代码中能想到的地方也做了一些保留，以供参考
- 各模块 templates/bak 目录下为html未添加bootstrap3样式之前版本

---
